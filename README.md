# Description

A sample repo showing how to use a Circle API SDK generated by OpenAPI generator (`typescript-node`)


# Instructions

1. Run the dependency installation script
    ```
    source setup.sh
    ```

2. Rename `.env.template` to `.env` in the `demo-scripts` folder and copy paste your bearer token in the indicated region

3. From the `demo-scripts` folder run `index.ts`

    ```
    npx ts-node index.ts
    ```
